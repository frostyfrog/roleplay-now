About RP Server

RP Server is an open-source web application that allows anyone to conduct a private text-based roleplay with their friends.

Technologies

The RP Server back-end is built on Apache, PHP, and MySQL: in other words, a LAMP server.

Additional libraries are listed below:

  • Slim, a PHP micro framework that helps you quickly write simple yet powerful web applications and APIs.
  • Twig, the flexible, fast, and secure template engine for PHP.
  • jQuery, a fast, small, and feature-rich JavaScript library.
